## About Me

I'm a Computer Engineer focused primarily on backend development, system architecture, and cloud-native applications.

I enjoy building logic-heavy systems involving REST APIs, authentication, real-time communication, background jobs, caching, database design, monitoring, and third-party integrations.

My primary experience is with **TypeScript, Node.js, NestJS, PostgreSQL, Redis, and Docker**. I am also expanding my backend expertise with **Java, Spring Boot, Maven, Flyway, and Spring Modulith**.

* 🍎 Developing applications for the Apple ecosystem
* 🤖 Exploring AI-powered developer tools and automation
* 🌍 Open to backend and software engineering opportunities
